CubanCoin Ecosystem

CubanCoin uses the BEP20 standard, which works in a similar way to the ERC20 standard of Ethereum.BEP20 is a developer-friendly token standard that allows anyone to implement fungible digital coins or tokens on Binance Smart Chain,as a low-cost, high-speed alternative to Ethereum, backed by one of the world's largest crypto companies.Using this technology is achieved: -Reinforced security -Greater transparency -Greater efficiency and speed in operations -easy to implement
